### 全屏截图

```shell
shift+command+3
```



### 可以显示隐藏文件，文件夹，再按一次，恢复隐藏

```shell
Command+Shift+.
```



### finder下可以前往任何文件夹，包括隐藏文件夹

```shell
Command+Shift+G
```



### 复制当前文件路径

```shell
commend+option+c
```



### 查看文件路径

```shell
command + i
```

### vim命令

~~~shell
i 开始编辑 
esc 退出编辑 
:w 保存但不退出 
:wq  保存并退出 
:q   退出 
:q!  强制退出，不保存 
:e!  放弃所有修改，从上次保存文件开始再编辑命令历史
~~~



### 显示隐藏文件

~~~shell
defaults write com.apple.Finder AppleShowAllFiles YES;KillAll Finder
~~~



### 关闭隐藏文件

~~~shell
defaults write com.apple.Finder AppleShowAllFiles NO;KillAll Finder
~~~
